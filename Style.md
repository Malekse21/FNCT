# UI Style Guide: The Wool & Soil System
**Concept:** Circular Infrastructure / Material Traceability
**Themes:** Earthy, Technical, Tactile, Grounded.

---

## 🎨 Color Palette (High-Contrast / Non-Metallic)

| Role | Color Name | Hex | Context |
| :--- | :--- | :--- | :--- |
| **Background** | Wool Oatmeal | `#F5F2ED` | Main app background (soft, warm white) |
| **Surface** | Raw Parchment | `#EDE8E0` | Card backgrounds, secondary panels |
| **Primary** | Leather Umber | `#5D4037` | Main buttons, navigation, headers (Real leather) |
| **Accent** | Slate Steel | `#455A64` | Data charts, technical borders, icons |
| **Success** | Dry Sage | `#6B8E23` | Healthy plants, stabilized soil, verified batches |
| **Alert** | Clay Red | `#A0522D` | Erosion (Injiraf) risk, contamination alerts |
| **Text** | Charcoal | `#263238` | Primary legibility |

---

## 🔡 Typography (The "Precision Meets Nature" Pair)

- **Headings:** *Source Serif 4* 
  - Style: Semi-bold. Feels grounded, traditional, and authoritative.
- **Body & Data:** *Inter* or *IBM Plex Mono*
  - Style: Regular/Medium. Use the Monospace version for GPS coordinates, batch IDs, and weight measurements to give it a "Factory Receipt" look.

---

## 📐 Components & Geometric Logic

- **Borders:** `1.5px` solid. Use **Slate Steel** for technical areas and **Leather Umber** for call-to-action areas.
- **Corners:** `4px` (Tight/Industrial). Avoid large "bubbly" curves.
- **Shadows:** None. Use **Borders** or **Color Blocks** to create depth. This keeps the UI looking "flat" and material-focused.
- **Texture:** A 2% "Noise" or "Grain" overlay on the **Wool Oatmeal** background to make it feel like fiber.

---

## 📊 Data Visualization (Injiraf & Waste Stats)

- **Charts:** Use solid "Block" bar charts. 
- **Mapping:** Use Mapbox "Terrain" or "Light" styles. 
  - **Injiraf Risk:** Represented by **Clay Red** hatch patterns.
  - **Protected Zones:** Represented by **Dry Sage** solid fills.
- **Icons:** Use the *Lucide* or *Feather* icon sets. Thin strokes, no fills.
